# `data_prep_legacy_nc/` (Legacy North Carolina data preparation)

This folder contains the original NC-only data workflows.

Use this path to reproduce historical datasets and splits used in the legacy NC experiments.

## Included workflow types

- species occurrence query/filtering,
- NAIP image download,
- random and stratified train/val/test sampling,
- spatial CV split creation.

For current US-wide work, prefer `data_prep/`.
